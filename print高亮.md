格式为：

**\033[显示方式;前景色;背景色m** 

显示方式有：

|  方式  | 含义     |
| :--: | ------ |
|  0   | 终端默认显示 |
|  1   | 高亮显示   |
|  4   | 使用下划线  |
|  5   | 闪烁     |
|  7   | 反白显示   |
|  8   | 不可见    |

颜色说明：

|  颜色  | 前景色  | 背景色  |
| :--: | :--: | :--: |
|  黑色  |  30  |  40  |
|  红色  |  31  |  41  |
|  绿色  |  32  |  42  |
|  黄色  |  33  |  43  |
|  蓝色  |  34  |  44  |
| 紫红色  |  35  |  45  |
| 青蓝色  |  36  |  46  |
|  白色  |  37  |  47  |

例如：

```
\033[1;31;40m
```
